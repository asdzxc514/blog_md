Element.classList()
ex) test2.classList.toggle()

메소드정리
1. item( Number ) : 클래스 값을 콜렉션의 인덱스를 이용하여 반환함
2. toggle ( String [, force] ) : 클래스가 있으면 지우고, 없으면 만들고
3. contain ( String ) : 특정 클래스 값이 존재하는지 확인
4. replace( oldClass, newClass ) : 기존 클래스에서 새로운 클래스로 교체함
